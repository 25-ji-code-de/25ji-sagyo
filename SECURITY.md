## 报告漏洞

我们非常重视项目的安全性。如果你发现了任何安全漏洞，请**不要**直接在 GitHub Issue 中公开报告。

请通过以下方式联系我们：

- 在哔哩哔哩私信 bili_47177171806（https://space.bilibili.com/3546904856103196，UID: 3546904856103196）
- 或者在 GitHub 上向维护者发送私信（如果适用）。

请在报告中包含以下信息：

1. 漏洞的类型和严重程度。
2. 复现漏洞的详细步骤。
3. 任何相关的概念验证 (PoC) 代码或截图。

我们会在收到报告后尽快确认，并努力在合理的时间内修复漏洞。在修复发布之前，请不要公开披露漏洞细节。

感谢你帮助我们维护项目的安全！
